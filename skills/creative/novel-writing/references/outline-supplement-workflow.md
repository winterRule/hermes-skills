# 大纲缺失弧线补充流程

## 触发条件

当以下任一情况发生时，必须先补充大纲再创作：

1. 大纲文件覆盖范围不及当前弧线（如全书大纲只到250章，但需要写251-260）
2. 大纲有弧线名但无逐章标题和内容要点
3. 大纲章节编号与实际文件编号不对应

## 步骤

### 1. 检测覆盖缺口

```python
# 在大纲文件中搜索当前弧线的章节编号
# 如搜索 251|252|253 → 无结果 → 确认缺口
```

### 2. 读取所有已有章节

```python
# 扫描目录中该弧线范围的已有文件
# 读全文理解当前叙事位置和人物状态
# 提取已有章标题作为锚点
```

### 3. 推断缺失章标题和内容

从已有章标题之间的叙事空间推断：
- 251「民间武装」→ 254「少年队」：需要522「联防」523「识字班外」
- 254→257「妇女救亡」：需要525「送信」526「补给链」
- 257→259「众志成城」：需要528「名册之外」

### 4. 创建补充大纲文件

格式完全遵循主大纲：弧线细纲 + 隐喻对照表 + 核心冲突 + 逐章表格。

```
# X大纲补充_弧XX-XX.txt

## 第X弧线「弧线名」（XXX–XXX章）

**弧线细纲**：...

**隐喻对照**：
| 虚构事件 | 历史隐喻 |

**核心冲突**：...

| 章 | 标题 | 状态 | 内容要点 |
|----|------|:--:|------|
| 251 | 已有标题 | ✅ | 已有内容 |
| 252 | 推断标题 | ⬜ | 推断内容要点 |
```

### 5. 用户确认

展示补充大纲，等待用户确认或调整后再开始创作。

## 验证数据

**Arc 11-2 (251-260)**：补充大纲后创作5章缺失章，全部首稿达标（章均2,532 CJK，零章需扩充）。

**Arc 11-3 (261-270)**：补充大纲后创作5章缺失章，全部达标。

**Arc 12 (271-300)**：补充大纲后创作15章缺失章，9章需扩充。首稿章均1,100-1,400 CJK——原因：弧12已有奇数章标题与初始补充大纲标题不符，需要先读已有章校准补充大纲。

## ⚠️ 关键陷阱：补充大纲必须匹配已有章标题

**Arc 12实际案例**：初始补充大纲为281-300章设定了"联合参谋部→第一份情报→分工→物资流→识字班的盟约课→山崎的反制→第一次协同→伤兵→苏大姐的统计→暗河的潮"等标题。但实际已有奇数章标题为："联合参谋部→磨合→第一次联合作战→民间编入→正面战场→敌后→信任→联动→联合后勤→统一战线"。

**规则**：在创建补充大纲前，必须先 `ls` 扫描目录确认已有章的**实际**标题，然后基于实际标题推断缺失偶数章的桥接标题。不能假设已有章标题符合预期。此规则适用于所有继承弧线。

## 关联步骤

补充大纲后，继承章（已有章）的格式修复必须同步完成：
- 内部编号漂移修复
- ASCII引号→中文弯引号转换
- 章末标记编号对齐
