# 双弧线对比模板

当同卷内连续两条弧线均完成审计后，使用此对比表呈现卷主题在不同战场中的应用。

## 使用时机
- 同卷内两条弧线均已完成 Phase 0-3 审计
- 两条弧线共享同一卷的核心主题（如第三卷「双线作战→艰苦相持」）
- 用户在弧线完成后要求故事性分析

## 对比维度

| 维度 | 弧线N | 弧线N+1 |
|------|-------|---------|
| 地形 | 山·冻土 | 水·淤泥 |
| 核心武器 | 竹雷·拖曳雷 | 水雷·锯齿环 |
| 敌后战术 | 撬桥·堵排水 | 水下推卵石·锚桩爆破 |
| 民间支援 | 搬运工 | 舟队·桐油填缝 |
| 物码闭环 | 石板战报 | 船板战报 |
| 收束符号 | 全圈偏北 | 水中悬浮点 |
| 弧线主题 | 守住 | 水会自己流 |

## Arc 13-14 实例

### 北岭 vs 江南

| | 北岭 | 江南 |
|----|------|------|
| 地形 | 山·冻土 | 水·淤泥 |
| 核心武器 | 竹雷·拖曳雷 | 水雷·锯齿环 |
| 敌后战术 | 撬桥·堵排水 | 水下推卵石·锚桩爆破 |
| 民间支援 | 十七搬运工 | 序列003舟队·三年桐油 |
| 后勤创新 | 科目002-甲·盐·家父遗物 | 科目002-乙·井水·无期限 |
| 医学/环境 | 冻土低温抑菌 | 水媒细菌·七度恒温 |
| 收束符号 | 全圈偏北（向北=下一次） | 水中悬浮点（水不需要方向） |
| 弧线主题 | 守住——山会一直在 | 守住——水会流，但船在 |

### 对比分析要点
1. **地形决定战术**：北岭的冻土硬壳→竹雷埋在冻土下等机甲压；江南的浅水区→叉号标记架桥点+水雷守锚桩
2. **敌后升级**：北岭的破坏是物理性的（撬木板、堵排水沟）；江南的破坏是系统性的（堵锚桩→浮桥漂走→帆布变船帆→闭环）
3. **民间序列演进**：北岭的搬运是一次性的；江南的舟队编入序列003，建立可复用的编入标准
4. **物码体系进化**：北岭引入石板刻战报；江南引入船板刻战报——载体从"山的石头"变成"水的木头"
5. **收束哲学递进**：北岭「全圈偏北」——方向由山决定；江南「水中悬浮点」——水不需要方向，「水自己会守着」
