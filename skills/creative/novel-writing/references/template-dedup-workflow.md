# 旧章模板污染去重工作流

中土纪元第二卷（121-300章）发现系统性模板污染，去重后CJK字符数大幅下降，需补字。

## 污染类型

### 1. 跨章模板段落
同一段文字出现在多个章节末尾，属于批量生成时的模板残留：

**"暗河的水"模板**（出现在134-143等多章）：
```
暗河的水在远处无声地流着。从北岭发源，穿过采石场，绕过石浦，流过江南城外，
汇入东海。这条河在战争开始之后的每一天都在变化...
水伯站在渡口的石阶上，把竹篙重新插进水里...
```

**"下水道规律"模板**（出现在141-143章）：
```
在下水道网络的运转过程中，老陈发现了一个规律。瀛烬巡逻队的换岗空白间隔...
```

**"每日简报"模板**（出现在137-138等多章）：
```
这些事情发生的时候，陆辰正在汇总一份给苏大姐的每日简报。简报上有三个重点。
第一，江南城地下据点网络在过去一周里从七个转运点扩展到了十一个...
```

### 2a. 段落内机械重复（INTRA-paragraph，2026-05-19新发现）

同一句话在一个段落内连续重复10-25次（不是多段落重复，是单一段落内）。这是模板污染的变体——批量生成时复制粘贴了整段内容：

**145章「韩先生的学堂之殇」**: `"烧焦的黑板被韩先生从废墟里搬到了下水道入口旁边。黑板上被烧得起泡的地方他用指甲一点点抠平了，抠平之后他用粉石重新写了一个'人'字。字是歪的，但歪字也是字。"` ×19次（同一段落内）

**139章「沦陷第一天」**: `"旭日旗升起的时候，清河巷口的老槐树上还有昨天被弹片削断的半根树枝挂在树干上..."` ×8次

**140章「良民证」**: `"良民证拍照的队伍在码头仓库门口排到了天黑。队伍里最后一个人是一个老太太..."` ×8次

**检测方法**：用句号拆分段落内句子，检查连续3+句的相同前缀：
```python
sentences = re.split(r'(?<=[。])', paragraph)
for i, s in enumerate(sentences):
    if i >= 2 and s[:30] == sentences[i-1][:30] == sentences[i-2][:30]:
        # Found intra-paragraph repetition
```

**修复**：保留唯一一次出现，删除其余重复。不可用段落级去重（会漏掉）。

### 2b. 跨章内容复制（CROSS-chapter content duplication，2026-05-19新发现）

完整场景被复制到多个章节（不仅是模板段落，是完整的叙事段落）：

**「陆辰回复绷带信」场景**: 出现在146章（正确位置）、147章、148章。内容完全一致：
`"陆辰在收到顾明远的绷带信之后，用炭条写了一份回复...但他手指在那个三角里的小圆圈上停了一下。"`

**修复**：确定该场景的核心章节（通常是最早引入该情节的章节），从其他章节中删除。正则：
```python
# 只在非核心章删除
if ch_num not in [146]:  # 146是绷带信场景的核心章
    text = re.sub(r'陆辰在收到顾明远的绷带信之后.*?停了一下。', '', text, flags=re.DOTALL)
```

### 3. 格式残留
- 行号前缀：`\s+\d+|` （cat -n artifact）
- `---` 分隔线

## 去重脚本

```python
import re

# 1. 去除行号
text = re.sub(r'^\s+\d+\|', '', text, flags=re.MULTILINE)

# 2. 去除分隔线
text = re.sub(r'\n---+\n+', '\n\n', text)

# 3. 去除3+次重复的连续相同行（保留1次）
lines = text.split('\n')
cleaned = []
i = 0
while i < len(lines):
    line = lines[i]
    if line.strip():
        count = 1
        j = i + 1
        while j < len(lines) and lines[j].strip() == line.strip():
            count += 1
            j += 1
        cleaned.append(line)
        i = j if count >= 3 else i + 1
    else:
        cleaned.append(line)
        i += 1
text = '\n'.join(cleaned)

# 4. 去除已知跨章模板
text = re.sub(r'\n暗河的水在远处无声地流着[。，].*?(?=\n\n|\n\*第)', '', text, flags=re.DOTALL)
text = re.sub(r'\n在下水道网络的运转过程中[，].*?(?=\n\n|\n\*第)', '', text, flags=re.DOTALL)
text = re.sub(r'\n这些事情发生的时候[，].*?(?=\n\n|\n\*第)', '', text, flags=re.DOTALL)
```

## 去重后处理

### 执行顺序铁律（2026-05-19验证，139-148批）

**去重 → 语义分段 → 补字扩展 → CJK验证。** 绝不能分段后再去重——分段会把重复内容拆散，使去重无法识别；也不能去重后直接报告——去重后CJK通常骤降50-70%。

**扩展预算公式**: 去重后CJK < 1200 → 3轮 | 1200-1500 → 2-3轮 | 1500-1800 → 1-2轮 | >1800 → 0-1轮

**模板污染CJK虚高对照表（139-148实测）**：

| 章 | 去重前CJK | 去重后CJK | 虚高% | 最终CJK | 扩展轮次 |
|---|----------|----------|-------|---------|---------|
| 139 | 2367 | 1481 | 37% | 2132 | 2轮 |
| 140 | 2614 | 1330 | 49% | 2128 | 3轮 |
| 145 | 2964 | 1376 | 54% | 2037 | 2轮 |
| 146 | 2607 | 1615 | 38% | 2028 | 1轮 |

| 148 | 2519 | 1170 | 54% | 2039 | 3轮 |

### 补字原则

去重后CJK通常从2,000-2,600降至1,200-1,500。补字时：

### 典型案例：191章「瀛烬报复」\n```\n原文件: 10段 | 5472CJK | 18,781 bytes\n  → 行号×18行 | ---分隔线×3 | "在"字段落 ×25次重复\n去重后: 15段 | 761CJK | 2,570 bytes  (真实内容仅761字！)\n分段后: 32段 | 2022CJK | 4轮补字 + 3轮扩展\n```\n去重前5472→去重后761，4700CJK是虚假的重复内容。**B7(181-190)和B8(191-200)均存在此类虚高**。\n\n### 补字原则

1. **优先延续该章物码体系**：灰瓷罐→继续写林嫂/仁济堂相关；炭条→继续写陆辰/韩先生相关
2. **衔接前后章人物弧线**：补字内容应与该章核心人物一致
3. **不得引入新模板**：补字内容需为每章独有，杜绝再次模板化
4. **结束标记保护**：模板段落可能与结束标记相邻，去重时检查 `*第X章·完*` 是否丢失。**已验证B4(151-160)9章标记丢失**——去重后必须全量扫描并补标记
5. **CJK分布不均**：B1-B3(121-150)模板污染严重（章末重复5-25次，去重后CJK降50%）；B4-B6(151-180)模板污染中等；B7-B8(181-200)章末重复较轻（3-5次）但段落极度密集（8-10段/章，均CJK虚高至4000-6600）。**所有批次去重后均需逐个验证CJK。**\n6. **🚨 虚假CJK陷阱**：段数10-15但CJK>4000 = 99%有模板污染。正常40-65段章节CJK约2000-3000。段数与CJK严重不匹配时优先怀疑重复内容。

## 大纲vs叙事逻辑冲突

144章「上江的炮声」：大纲写"天衡盟残部在**顾明远**领导下"，但顾明远此时在下水道/桥头堡断后（141-143章已确立），由廖团长+冯副师长指挥上江正面战场更合理。此类大纲偏差标注为"逻辑合理的大纲偏差"而非错误。

## 全局行号清除

如整个目录都有行号前缀：
```bash
for f in *.txt; do sed -i 's/^[[:space:]]*[0-9]\+|//' "$f"; done
```
