# 小说创作永久强制规则

## 一、字数铁律
1. 全书总字数严格控制：150 万字整
2. 单章字数：**硬地板2000字**，可以超过3000字但不能少于2000字
3. 全书结构严格：共 5 卷，不得增减
4. 禁止水字、禁止重复、禁止凑字数、禁止省略关键情节

## 二、核心设定不可遗忘
1. 模式：双向穿越 —— 明朝皇帝 ↔ 现代影视城
2. 主线：现代学习知识 → 带回古代全面改造
3. 风格：喜剧碰撞 + 硬核权谋 + 写实改革 + 文化反思
4. 无系统、无空间、无异能、无玄幻，仅靠双向穿越 + 知识学习

## 三、内容方向不可偏离
1. 现代线：剧组生活、拍戏、查资料、学科技、搞图纸、收集工具
2. 古代线：朝堂制衡、吏治、军事、经济、民生、文化、教育全面改革
3. 必须覆盖：军事、经济、民生、文化、制度五大改造方向
4. 改革必须循序渐进，符合历史逻辑，不一键工业化、不降智、不龙傲天

## 四、输出格式强制要求
1. 每章必须完整、有头有尾、有冲突、有推进
2. 语言符合人物身份：皇帝古风庄重，现代口语自然
3. 剧情双线并行，现代与古代来回切换，节奏稳定
4. 每次生成内容必须先自查：字数、设定、主线、逻辑是否合规

## 五、写作节奏
- 每个弧线10章左右，连续创作
- 写完后统一补足短章节（补齐pass）
- 保持合理性，有创造性
- 按照大纲进行开发
- 字数可以超出但不得少于2000字

## 六、单章固定写作模板
### 开篇钩子 (300-500字)
- A类：悬念钩子 - 上一章结尾危机/转折/疑问的延续
- B类：场景切入 - 直接进入一个生动的场景
- C类：心理独白 - 主角对当前处境的感慨/判断/决策

### 发展 (1200-1800字)
- 古代章：60%剧情推进 + 20%知识应用 + 20%冲突展示
- 现代章：40%喜剧/文化冲突 + 30%学习成长 + 20%人际互动 + 10%准备/收集

### 高潮/转折 (300-400字)
每章必须有一个"小高潮"

### 结尾 (200-300字)
点题 + 悬念 + 过渡

## 七、写作风格铁律
1. 古代戏份语言：主角内心/对白——半文半白，简洁有力
2. 现代戏份语言：自然口语
3. 朝堂对话：简练、有锋芒，避免长篇大论
4. 学习场景：知识点嵌入剧情，不单独罗列
5. 严禁外貌描写超过50字/次
6. 严禁内心独白超过200字/段
7. 每章至少出现一次"动作推进"
8. 全篇视角：第三人称有限视角（始终跟随主角，不切换POV）
