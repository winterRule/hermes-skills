# 稀疏大纲卷创作模式

适用于《中土纪元》——当大纲中某个卷/弧线的描述不超过一页、只列出弧线名称和高级主题、无逐章分解，或大纲在该范围完全缺失（如arcs 9-16在v4大纲中全缺）。

**前置必做**：如果大纲有章节条目缺失，先按 `references/outline-gap-supplement.md` 补充大纲——不要在没有完整大纲的情况下开始创作。用户明确要求"大纲缺失先补充大纲"。

## 判断标准

当大纲中某个卷的描述不超过一页、只列出弧线名称（如"北岭会战、江南保卫战、隘口血战…"）和核心主题、没有"第X章｜标题：内容"格式的逐章分解时，触发此模式。

## 弧线规划步骤

### 1. 提取主题
从大纲摘要中提取该卷的所有弧线名和主题方向：
```
第三卷: 双线作战 → 艰苦相持 → 局部反攻 → 胜利
弧线: 北岭会战、江南保卫战、隘口血战、中原大会战、百团破袭、反攻胜利
```

### 2. 映射历史原型
将弧线名映射到真实历史战役/事件，理解叙事功能：
- 北岭会战 → 淞沪会战（首次大规模正面防御）
- 江南保卫战 → 南京保卫战（首都沦陷后的坚守）
- 百团破袭 → 百团大战（敌后大规模反击）
- 隘口血战 → 武汉会战（战略相持转折）
- 中原大会战 → 长沙会战（反复争夺）
- 反攻 → 湘西会战/日本投降

### 3. 创建10章逐章大纲
每弧线分解为10章，章名遵循联合参谋部纪元体系：
- 第1-2章：战前准备/情报/集结
- 第3-5章：战斗主体（正面+敌后双线）
- 第6-7章：危机/转折（物资/伤亡/敌后破袭）
- 第8-9章：百姓支前/民间序列
- 第10章：战后收束+物码闭环+下一弧线钩子

### 4. 确保连续性
- 物码体系延续：前弧线的石子/归码/枯枝/蓝色/账本等必须在本弧线出现并演进
- 人物弧线不中断：七核心（陆辰/萧旅长/江寒/铁柱/钱伯钧/沈之默/阿来）每弧线至少出场一次
- 苏大姐石浦线/韩先生识字班线/阿禾码头石子线 每弧线至少各推进一次
- 时间线标注清晰：从联合参谋部成立起算月份（如"第五个月"、"第七天"）

### 5. 首稿字节目标
稀疏大纲弧线没有现成的逐章关键词可对标审计——因此首稿质量更加关键。严格遵循8000+ bytes目标，3章一批验证。

## 弧线终章收束模式

每弧线最后一章必须含：
1. 阿来石壁"每日存在"编年史新增一行
2. 苏大姐日志一句话总结（通常极短，1-2个词）
3. 韩先生识字班教一个新字或拆字
4. 阿禾在码头多放一颗石子（第N弧线=第N+2颗石子，从第一弧线两颗起算）
5. 一个物码闭环（如弹片→船钉、盐→科目002、石板→枯槐树下）

## Arc 13 成功案例

Arc 13（北岭会战）按此模式创作，结果：
- 8/10章首稿达标（仅127/130需单轮微扩）
- 章均2,428 CJK字
- 大纲关键词38/40（95%）
- 物码体系完整闭环：石板刻战报→枯槐树下，石子四颗→偏北的点，科目002-甲盐→家父遗物

## Arc 14 成功案例

Arc 14（江南保卫战）按此模式创作，结果：
- 7/10章首稿达标（137/138/139需单轮扩充，各~200-300 CJK追加）
- 章均2,231 CJK字
- 大纲关键词39/40（97.5%）
- 地形转型：从北岭「山·冻土」→江南「水·淤泥」，战术体系从竹雷拖曳雷→水雷锯齿环
- 物码闭环创新：船板（遗物船弹孔边缘渗三年桐油）替代片麻岩为战报载体
- 新序列：民间舟队序列003建立（和北岭的搬运工形成一次性vs制度化对比）
- 新科目：002-乙「井水·无期限」——将人道主义供给编入联合后勤体系
- 第一处非战斗收束：碾米厂四十二人放下桨→竹筒井水→「渴」字教学——全书首次以围而不歼收束一场战斗
