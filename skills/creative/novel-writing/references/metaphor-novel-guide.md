# 隐喻历史小说创作指南

## 定义

隐喻历史小说是历史穿越小说的一个子类：主角魂穿进一个架空大陆，该大陆的势力格局、历史事件、人物群像是真实历史（通常为1911–1949近代史）的隐喻映射。主角的唯一金手指是"知道历史走向"——他在现代学到的历史知识成为他在架空世界中的战略优势。

## 核心特征

### 1. 单向穿越，不可返回
- 不同于双向穿越（古代↔现代），隐喻小说是魂穿进入架空世界后不再返回
- 现代仅以"前夜记忆/梦中闪回"的形式穿插出现（不超过全书5章）
- 主角的心理锚点：从"想念现代"到"此心安处是吾乡"

### 2. 势力格局的隐喻映射
架空世界的每一方势力都有明确的历史原型，映射关系必须在创作前明确定义：

| 架空势力 | 历史原型 | 特征 |
|----------|----------|------|
| 赤炬会 | 共产党 | 知识分子+工农联盟，根据地建设，群众路线 |
| 天衡盟 | 国民党+北洋军阀 | 最大军事政治联盟，内部腐败分裂 |
| 瀛烬帝国 | 日本 | 海岛工业强国，扩张侵略 |
| 伪衡政权 | 伪满/汪伪 | 侵略者扶植的傀儡 |
| 星火盟 | 民间抗日力量 | 自发抵抗，游击队 |
| 玄朔部族 | 满清残余 | 旧王族，腐朽覆灭 |

### 3. 科技/力量体系映射
- **遵循prompt的术语体系**：每部隐喻小说的科技映射术语由用户prompt定义，不可套用其他小说的设定。例如：某部小说用「灵械术」「灵晶」映射工业技术，另一部可能直接用「机甲」「能量武器」。读取prompt中的势力设定关键词，以此为唯一权威来源。
- 力量体系梯度映射真实世界的技术发展阶梯（民用→军事）
- 通讯手段映射：灵讯塔/电台≈无线电报，传音石≈电报机，飞鸽传书≈原始信使
- 不引入超自然魔法——保持「硬核历史」的质感

### 4. 主角成长弧
- **第一卷**：旁观者→情报分析员（利用现代历史知识解读架空世界格局）
- **第二卷**：战术参与者→战略参谋（将现代组织管理知识转化为架空世界的制度设计）
- **第三卷**：前线经历者（亲历战争，从"知道伤亡数字"到"认识每一个零背后的人"）
- **第四卷**：全局观察者（见证战后势力洗牌，做出道德抉择）
- **第五卷**：见证者→功成身退（新世界建立后回到最初的地方）

### 5. 叙事规则
- **禁止上帝视角预知**：主角不能直接说"历史会这样走"，只能通过情报分析间接预判
- **禁止直接点名原型**：书中绝不出现"毛泽东""蒋介石"等真实人名，也绝不出现"中国""日本"等真实国名
- **底层视角优先**：主线事件（大会战、建国）通过小人物视角呈现——识字班的女孩、送粮的老汉、被救的伤兵
- **每章必须有一个"动作推进"**：不是议论、不是抒情，是有人做了一件事

## 与双向穿越小说的区别

| 维度 | 双向穿越（横店类） | 隐喻小说（中土类） |
|------|-------------------|-------------------|
| 穿越方向 | 双向，频繁来回 | 单向，一去不回 |
| 金手指 | 现代知识+物品搬运 | 仅有"知道历史走向" |
| 核心冲突 | 古代制度 vs 现代知识 | 各方势力之间的历史性对抗 |
| 现代线占比 | ~20% | <5%（仅闪回） |
| 语言风格 | 古代半文半白+现代口语 | 架空世界的统一语体（热血动漫风） |
| 结构 | 4-5卷，古今交替 | 5卷，线性推进 |
| 结局 | 皇帝改革成功 | 主角功成身退，回到起点 |

## 创作陷阱

1. **过度说教**：不要让主角站在道德高地上对架空世界的人物说教——他只是一个"知道得更多"的观察者，不是一个全知全能的导师
2. **隐喻穿帮**：不要让读者一眼看出"这不就是XXX吗"——隐喻应该像果核藏在果肉里，吃的时候能尝到味道但看不见形状
3. **英雄史观**：不要把历史变革归因于主角一个人的决策——真正的推动力应该来自无数普通人的集体行动（识字班、独轮车、根据地）
4. **淡化残酷**：战争不是热血动漫——每一场胜利背后都有具体的牺牲，要写，但不要消费苦难
