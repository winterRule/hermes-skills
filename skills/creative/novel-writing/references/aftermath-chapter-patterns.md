# Aftermath Chapter Patterns (战后收束章节创作模式)

**Arc 16 validated** (401-410, 10 chapters). Aftermath chapters — covering returns, reflections, first peaceful acts, character farewells — consistently produce **1200-1600 CJK first drafts** when planned with 3-4 vignettes. The same 3-4 vignette count that works for battle chapters (2000+ CJK) fails here because:

1. **The tone is quieter**: Without combat action, each vignette "feels" complete sooner — but the word count requirement is identical
2. **Single-character focus**: Aftermath chapters often follow one character (焦老农犁地, 铁柱铸犁, 阿来刻好, 老陈送书) — fewer parallel action lines to weave together
3. **Reflective density vs. action density**: Interior monologue and sensory description take fewer CJK characters to convey than tactical sequences

## Pre-writing Checklist for Aftermath Chapters

Before writing any chapter in the "quiet/aftermath" class:
- [ ] Plan **5-6 vignettes** (not 3-4)
- [ ] Include at least one vignette where the character interacts with another character (not just solo reflection)
- [ ] Include at least one vignette with a **physical transfer** — an object passing between hands (not just observation)
- [ ] Include at least one vignette bridging to a location or character from a _different_ arc (跨弧衔接)

## Expansion Budget

Budget **2-3 rounds** of expansion per aftermath chapter. The batch expansion pattern (`execute_code` with multiple `patch` calls) works but expect to run it multiple times:

- Round 1: Add substantive closure paragraphs (500-800 CJK each) — chapters go from ~1400 to ~1800
- Round 2: Add bridging vignettes with other characters (250-400 CJK) — chapters go from ~1800 to ~2000
- Round 3: Targeted pushes (100-200 CJK) for stragglers

## Technique: Pre-plan Vignettes with Objects

Aftermath chapters benefit from object-based transitions. Plan a physical object that moves through the chapter:

- 406: 石英岩 → 凿刀平放 → 淤土撒字 → 诊所窗台
- 407: 犁铧 → 收帆结缰绳 → 蚌壳苗
- 408: 铸铁 → 泥模 → 犁头铸纹 → 刨刃 → 余铁
- 409: 弹痕灯罩 → 囤船捻灰 → 课本 → 伤残兵拐杖
- 410: 拐杖 → 青石板 → 旧信碎片 → 教室

Each object provides a "hook" for adding another 200-300 CJK of sensory/tactile description.
